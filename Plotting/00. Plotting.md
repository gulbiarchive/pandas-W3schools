A scatter plot needs an x- and a y- axis.

In the example below we will use â€˜Durationâ€™ for the x-axis and â€˜Caloriesâ€™ for the y-axis.

# Histogram

In the example below we will use the â€˜Durationâ€™ column to create the histogram:

